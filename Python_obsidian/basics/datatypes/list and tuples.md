**List** → an **ordered, mutable (changeable)** collection datatype.
```
my_list = [1, 2, 3, "hello"]
```

**Tuple** → an **ordered, immutable (unchangeable)** collection datatype.
```
my_tuple = (1, 2, 3, "hello")
```
